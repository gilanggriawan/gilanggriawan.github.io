EMAS Web Design & Web Development

Client: EMAS

Year: 2022

EMAS is an Agency that provides Digital Marketing services for business people. This agency was only founded this year, 2022. And this is their first website.

Goals

This is what EMAS wanted to achieve on their first site.

  1. Brand Awareness. They want to have value in their services in digital marketing.

  2. Clean and Modern. They want a clean and modern look.

The User & Audience

The target audience is people who are not familiar with EMAS, especially entrepreneurs who want to use its services.

Role & Responsibility

  1. UI Designer. I gathered some necessary information to implement into the design, create user flows, wireframes, visual designs, and prototypes.

  2. Project Manager. I communicated and discussed with them to work on this project.

  3. Wordpress Developer. I also have basic frontend skills, they asked me to develop this website using Wordpress, so it was quite easy for me to improvise on this work.

Scope

This is their first website, not many pages they want to show. There are only two pages;

  1. Home

  2. News

Process

Gathering Requirement

As I explained above, they want a clean and modern website. So I tried to ask about the color or website reference they want.

Design Reference

These are some of the reference sites they like.

Sitemap

They want their website to focus on one page to showcase their services, so the sitemap is very simple. This is a sitemap.

Wireframe

The next process is to create a wireframe.

Struggle

First, I tried to follow the EMAS Brand Identity, namely by using the nuances of Gold in the design, it was quite difficult for me to get an attractive but clean design composition.

Second, I had to find some additional illustrations to give this design an interesting impression.

Third, I worked with a Copywriter to create this design, and one of my problems was when I wanted to make some cards to showcase EMAS Services, but the description of each service had long text.

Visual Design

Yeah, this is the most fun process, Visual Design. After getting the Design Elements and Color Palettes, I created a Visual Design. And this is a visual design of the site.

Design Decision

Hero Section

I made a color that gives the impression of Gold on the main title in the hero, this is one way I follow the EMAS brand identity.

Slider in Hero

I think it's a good idea to use the slider that contains the illustration service, I worked with a Graphic Designer on the illustrations used here. And since I'm also a developer, I can also implement this design.

The Cards

As I explained above, I want to use the card to display services from EMAS. To display a long description, I made this card dynamic on hover. Because I'm also a developer, so I'm sure this can be implemented.

Black Gradient

I made a black gradient on the hover card, testimonial and footer. Because the black color also gives the impression of luxury.

Use of Ilustrasion

In the How to order and Clients section I thought I should make this more interesting. So I'm using some flat illustrations here.

Result

After discussing with the client, they felt that this design was very good and interesting, so the next step was they asked me to develop this design using Wordpress.

I've never used Wordpress before, but I have a basic interface like HTML and CSS so it's pretty easy for me to adapt to this project.

Initially I was asked to use Elementor Builder to develop this website, so I tried to learn Elementor, but in the middle of the process, for some reason they asked me to use Oxygen Builder. But I don't mind because as I explained above that I have a basic frontend, so that's enough. easy for me to work on this project. And this is a website that I built using Oxygen Builder.
